# 坦克大战 (Tank Battle)

一款使用纯 HTML5 Canvas 开发的经典坦克大战网页游戏。

## 玩法

- **移动**: 方向键 (⬆⬇⬅➡) 或 WASD
- **射击**: 空格键
- **暂停**: P 键
- **重新开始**: R 键

## 游戏特色

- 多层关卡，敌人数量和种类随关卡递增
- 三种敌人类型：普通坦克、快速坦克、重型坦克
- 可破坏的砖墙和不可破坏的钢铁墙
- 生命系统和得分系统
- 流畅的爆炸特效

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

```
open index.html
```

或使用任意静态文件服务器：

```
npx serve .
```
